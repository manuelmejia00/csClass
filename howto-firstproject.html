Manny's How-To Page

Introduction

In this project, I learned about HTML and CSS, which are languages used to create a webpage. Like I said on my About Me page, I learned that HTML is used to create the makeup of a page, while CSS styles the HTML and makes the webpage look pretty.My webpage has several components. The code for all these components are located in three files of mine: the index.html file, which contains all of the HTML for my page, the navbar.css file, which styles my navigation bar, and the index.css file, which styles everything else on my page.

Headers

To create the headers on my page, I used the <h1> and <h3> tags. The two tags are identical, but the <h1> tag creates a bigger heading. To use these tags, I simply wrote what I wanted the heading to say, whether it be "Who am I?" or "What have I learned in terms of programming?" I then gave each of the headings an id by using the id attribute, which, as I mentioned above, gives the tags a unique name that I can use to call upon them later. After that, I went to my index.css class to style the headings. For all of the <h3> headings, I gave them a unique background color by calling them by their individual ids and then writing "background-color: color" , where "color" was whatever color I gave them. For the lone <h1> tag of my page, I used something a little different. First, I colored the header's text by going into the index.css file, calling the header, and writing "rgb:(102, 102, 255)" -this uses RGB coloring, a way to specify colors- , which gave the header's text a nice blue color. And, instead of giving the <h1> tag a background color, I gave it a border. Underneath the rgb statement, I wrote "border-top-style: dotted" and "border-bottom-style: dotted". Doing this gave the <h1> header a dotted border above and below of the text.

The Navigation Bar

At the very top of my page, there is something called a navigation bar, which allows users to more quickly jump to sections of my page. To make the navigation bar, I had to create a few things. Firstly, I made a CSS file for the navigation bar. Making this file would allow me to later change the style of the navigation bar. After this, I went to my main index.html file and created the actual navigation bar using the <ul> and <li> tags, which stand for unordered list and list item, respectively. I used these tags, because the navbar is essentially a list, with each of the navbar buttons being a list item. With that, I made four list items, for each section of my page. I also added <a> tags along with the <li> tags. The <a> tag is the anchor tag and it's used to make links with the href attribute. So, I linked each of the navbar list items to different sections of my page, by writing "href=#section", where #section was the id (which gives tags a unique name, that can be called upon in other places) of whichever section of the page I was linking to.



The Paragraphs

To make all of text for the paragraphs on my page, I used the <p> tag. As you can see in the following images, the <p> tag is a very simple tag that allows me to write text on my page. To then make all of the text for the paragraphs centered, I had to use some CSS. I realized that I actually wanted everything in my page, or at least my page's body, to be centered. So, I called all of the tags in my body by writing body, and then I wrote "text-align:center", which aligned all of my pages text to the center. I also set the font to be arial, and added an animation that I talk about in the "Animations" section of this page. After I did this, I realized that I wanted each of my paragraphs to have a colored border, but I wanted the paragraphs to have a uniquely colored border. To make each of my paragraphs unique, I had to add an id attribute. to each of their p tags. I gave them ids like "first-paragraph." Then, I went to my index.css file and called each of the paragraphs one by one, using their ids. I gave them borders by writing "border:" and then specifying how big I wanted the border to be, what kind of border I wanted, and the color of border. The end result looks like "border: 5px solid lightpink" , which gives me a solid light pink border that is 5 pixels big.

The Images

In order to put all of the images on my page, I had to use the <img> tag. To display an image, I had to give the <img> tag a source attribute, which is written as src. For me, I usually uploaded a picture to an image hosting website, and then I used the link it gave me as the source for my images. I gave all of my images borders that matched the paragraphs they were related to. I did this in the exact same way I did with the paragraphs, by calling the images by their id in the index.css file, and then creating a border, for example "border: 5px solid lightpink".

The Animations

There are a few animations that are found on my page. The first one is the hover animation, which affects or changes an element when your mouse hovers over it. On my About Me page, this animation can be seen in two places. The first is when you hover your mouse over the picture of me sitting on a rock; the picture enlarges. To create this animation, I had to define the dimensions of the image before the mouse hovers over it, and the dimensions after it hovers over it. To do this, I went to my index.css file and called the picture by the id I had given it, "manny-rock". After calling it, I specified its beginning dimensions by writing how many pixels I wanted the height and the width to be. Beneath that, I wrote the name of the animation and specified that I wanted it to last for 2 seconds. I then wrote a second piece of code that said what I wanted to happen when the mouse hovers over the picture; increase the dimensions. I applied a similar hovering animation to the headers of my page, except instead of increasing the dimensions, I made it so that hovering over the headers changes the headers' background color to light green. A second animation on my page is the background color change. Here, I had to make the animation using something called "@keyframes", and I made the background alternate between two colors over a span of 7 seconds.
